\# AI Study Platform 🚀



An Enterprise-Level Full Stack Application built using:



\- Frontend (Web UI)

\- Java Spring Boot Backend API

\- Python AI Microservice

\- MongoDB Database



---



\## 📌 Project Structure



apps/

├── web-frontend

├── backend-api

└── ai-service



---



\## 🛠️ Tech Stack



Frontend:

\- HTML, CSS, JavaScript



Backend:

\- Java

\- Spring Boot

\- REST API



AI Service:

\- Python

\- FastAPI / Flask



Database:

\- MongoDB



---



\## 🔐 Security



This project follows enterprise-level security practices:



\- All `.env` files are ignored

\- API keys are NOT stored in GitHub

\- MongoDB credentials are NOT public

\- Virtual environments are ignored

\- node\_modules are ignored



---



\## ▶️ How to Run



\### Frontend

cd web-frontend

npm install

npm run dev



\### Backend

cd backend-api

mvn spring-boot:run



\### AI Service

cd ai-service

python -m venv venv

venv\\Scripts\\activate

pip install -r requirements.txt

python app.py
